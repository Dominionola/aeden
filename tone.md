## QUICK REFERENCE GUIDE

### When Writing UI Copy:

**DO:**
- Use contractions (you're, we're, can't)
- Start sentences with "You" when relevant
- Use active voice
- Keep it short (under 15 words when possible)
- Be specific ("5 posts" not "some posts")
- Use real numbers, not ranges
- End with periods, not exclamation marks

**DON'T:**
- Use emojis in serious contexts (errors, settings)
- Add "just" or "simply" (patronizing)
- Say "please" in buttons
- Use "very" or "really" (weak words)
- Write in third person
- Use passive voice
- Add filler words
